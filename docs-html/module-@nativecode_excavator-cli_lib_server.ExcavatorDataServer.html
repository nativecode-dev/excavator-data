Class: ExcavatorDataServer

@nativecode/excavator-cli/lib/server.ExcavatorDataServer()

A default API server that can be used to quickly update the database via REST calls. It is recommended that you use another package, such as ExpressJS. However, that shouldn't preclude you from using this in production, because it's just a basic node http server.
